🌮 Sabor Mexicano

Sitio web diseñado para presentar una experiencia visual y atractiva de la gastronomía mexicana. Este proyecto busca resaltar platillos tradicionales mediante un diseño moderno, responsivo y fácil de navegar.

- Demo en vivo

👉 https://damian7777777777.github.io/Sabor-Mexicano/

🛠️ Tecnologías utilizadas
HTML5
CSS3
GitHub Pages (para despliegue)

✨ Características
Diseño responsivo (adaptable a móvil y desktop)
Interfaz atractiva enfocada en comida mexicana
Navegación sencilla e intuitiva
Secciones organizadas (menú, información, contacto, etc.)

🎯 Objetivo del proyecto

Este proyecto fue desarrollado como práctica de desarrollo web para mejorar habilidades en diseño frontend y despliegue de sitios estáticos.

📌 Posibles mejoras
Integración con backend (reservas o pedidos)
Conexión a WhatsApp o formularios reales
Animaciones más avanzadas
Optimización SEO
